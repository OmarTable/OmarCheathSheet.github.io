HTML Table Refrence

Table Tags

Tag Name Description
<table> Table The wrapper element of all HTML tables.
<thead> Table Head The set of rows defining the column headers.
<tbody> Table Body The set of rows containing table data.
<tr> Table Row The table row container.
<td> Table Data The table row container.
<tfoot> Table Footer The set of rows defining the footer in table.

Table Attributes

Attribute Name Description
colspan Column Span Defines how many columns a td element should span.
rowspan Row Span Defines how many rows a td element should span.
